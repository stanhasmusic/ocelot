# ocelot
ocelot is a 2D schump arcade style game built in godot by a noob. 

this is a learning project, so it will be rough around the edges. 

the goal is to create a full game loop with 5 levels and fun, challenging gameplay. 

the game uses free-to-use assets (add source here). 

it started out in antigravity using Gemini to code, then moved to VS code and Claude Code.
